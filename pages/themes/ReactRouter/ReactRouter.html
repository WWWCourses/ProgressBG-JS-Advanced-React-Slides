What is routing and why?

Overview

Routing is the process through which the user is navigated to different pages on a website
Client-side vs Server-side Routing

Routing in React

React doesn't provide native routing support, but the React Router library is commonly used.
Reference: Route-based code splitting @reactjs.org
Installation:

				npm install react-router-dom@6

Example


			import React from "react";
			import { BrowserRouter, Routes, Route, Link } from "react-router-dom";
			import { About, Contacts } from "./routes/pages";

			export default function (params) {
				return (
					<BrowserRouter>
						<nav>
							<Link to="/about">About page</Link>
							<Link to="/contacts">Contacts page</Link>
						</nav>
						<Routes>
							<Route path="/about" element={<About/>}></Route>
							<Route path="/contacts" element={<Contacts/>}></Route>
						</Routes>
					</BrowserRouter>
				)
			}
