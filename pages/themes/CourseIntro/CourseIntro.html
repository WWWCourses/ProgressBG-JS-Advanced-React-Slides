What you'll learn

What you'll learn/gain from the course

Thorough understanding of the JavaScript's language through many real-world examples.
Everything about Functions and Functions as first-class citizens
Basic concepts of functional programming paradigm
Thorough understanding of JavaScript's objects and the prototypal inheritance
Get familiar with JavaScript design patterns
that will help you understand the "magical" concepts like Observables, Mixins, and etc.
Write effective and easy to maintain code implementing the JavaScript's best-practices

Web Services, AJAX, SPA
